# PO-2.4 Salimov Alikhan

Repository with classroom C# tasks split into separate numbered folders.

## Tasks

- `Task1_FindMaximum` - find the maximum number in an array.
- `Task2_SumArray` - calculate the sum of array elements.
- `Task3_EvenNumbers` - print only even numbers.
- `Task4_CountPositiveNumbers` - count positive numbers.
- `Task5_ReverseArray` - reverse array output.
- `Task6_AverageValue` - calculate the average value.
- `Task7_SearchElement` - search for an element in an array.
- `Task8_MultiplicationTable` - print a multiplication table.
- `Task9_FactorialMethod` - calculate factorial with a method.
- `Task10_MinMaxMethods` - find minimum and maximum using methods.

Each task is a separate console project and can be started with:

```powershell
dotnet run --project .\Task1_FindMaximum\Task1_FindMaximum.csproj
```
